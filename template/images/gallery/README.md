# Image Placeholders

This folder contains placeholder images that need to be replaced with your actual project photos.

## Required Gallery Images

You need to add 12 gallery images (or adjust the gallery.html file to match your number of images):

- `project-1.jpg` through `project-12.jpg`
- Recommended size: 800x800px (square format)
- Format: JPG
- Max file size: 500KB each (compress with TinyPNG.com)

## Where to Get Images

1. **Your Own Projects**: Use photos from your completed roofing projects
2. **Stock Photos**: 
   - [Unsplash](https://unsplash.com/) - Free high-quality photos
   - [Pexels](https://www.pexels.com/) - Free stock photos
   - Search for: "commercial roofing", "flat roof", "roofing construction"

3. **Hire a Photographer**: For best results, hire a professional to photograph your work

## Image Tips

- Use consistent lighting and angles
- Show before/after comparisons
- Include team members working (with permission)
- Highlight quality workmanship
- Show variety of project types
- Ensure images are sharp and well-lit

## Quick Setup

To quickly populate the gallery with placeholder images:

1. Find 12 roofing-related images online
2. Resize them to 800x800px
3. Name them project-1.jpg through project-12.jpg
4. Place them in this folder
5. Refresh your website

---

**Note**: The gallery.html file currently references 12 images. If you have more or fewer images, update the gallery.html file accordingly.
